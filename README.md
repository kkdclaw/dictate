# dictate

Локальная push-to-talk диктовка для macOS (Apple Silicon). Зажал правый Option —
говоришь — отпустил — текст вставился в активное поле. Полностью офлайн.

Пайплайн: микрофон → Whisper large-v3-turbo (MLX, словарь терминов через
initial_prompt) → при необходимости LLM-чистка слов-паразитов (Qwen3-4B, MLX) →
вставка через буфер + ⌘V → история в SQLite.

## Установка

```bash
uv sync
cp terms.example.txt terms.txt   # свой словарь терминов
./daemon.sh install              # демон + автозапуск при входе; разово: uv run dictate.py
```

При первом запуске скачиваются модели с HuggingFace (~4 ГБ). Недостающие
разрешения (Микрофон, Универсальный доступ, Мониторинг ввода) приложение
запрашивает само системными диалогами — выдай их процессу `.venv/bin/python3`
и сделай `./daemon.sh restart`. Статус виден в начале `dictate.log`.

## Управление

- **Push-to-talk:** зажми правый Option, говори, отпусти — текст вставится.
- **Toggle-режим (длинная диктовка):** короткий тап Option (<0.35 с) — запись
  идёт без удержания; второй тап — стоп и вставка.
- **Esc** во время записи — отмена без вставки.
- Иконка в меню-баре: ⏳ прогрев → 🎤 готов → 🔴 запись. В меню — последние
  фразы (клик копирует), переключатель LLM-чистки, словарь и лог.
- `./daemon.sh install|restart|uninstall` — управление launchd-агентом.

## Особенности

- Словарь `terms.txt` подсказывается Whisper прямо при распознавании — главный
  инструмент качества, дописывай туда всё, что диктовка перевирает.
- LLM-чистка запускается только если в тексте есть слова-паразиты (эээ, ну...).
- Если активный микрофон отдаёт тишину (AirPods в кейсе, крышка закрыта) —
  скрипт сам находит живой микрофон и защищается от галлюцинаций Whisper.
- История: `history.sqlite3` (чистый текст, сырой текст, длительность,
  приложение-получатель).

## Замеры (MacBook M5, 16 ГБ)

- Чистая фраза: ~1 с от отпускания клавиши до вставки.
- Фраза с паразитами: ~1.7 с (+LLM).
- Русский + английские термины распознаются на уровне выше диктовки Apple.
